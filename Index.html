Website Embedder with Brightness Control

Open any website in a new tab with fullscreen about:blank embedding

Example: https://www.wikipedia.org or google.com

How to Use:

  • Enter a website URL above and click "Open in New Tab"
  • The website will open in a new tab with about:blank embedding
  • Once the website is loaded, press Ctrl + Shift + Alt simultaneously to open the brightness control panel
  • Adjust brightness using the slider or by entering a value in the number box (even negative values work)
  • Press Ctrl + Shift + Alt again to close the control panel
  • You can also close the panel using the X button
Note: Some websites may block embedding for security reasons. If a website doesn't load properly, try a different site.
